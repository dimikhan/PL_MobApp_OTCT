#JazzHub bookmarks toolchain

Use this toolchain template to document existing tools available in some [JazzHub project](https://hub.jazz.net).

To get started, click **Create toolchain**.

[![Deploy To Bluemix](https://new-console.ng.bluemix.net/devops/graphics/create_toolchain_button.png)](https://new-console.ng.bluemix.net/devops/setup/deploy/?repository=https%3A//github.com/hmagph/jazzhub-project-template)

For more information about toolchains, see [Custom toolchains in one click with IBM Bluemix DevOps Services](https://developer.ibm.com/devops-services/2016/06/16/open-toolchain-with-ibm-bluemix-devops-services/).
